-Litro Keyboard-
　　シンプル音楽編集ソフト


●インストール
適当なフォルダに圧縮ファイルを解凍してインストール完了です。
その他にインストールするファイルやライブラリは特にありません。
アンインストールは下記のファイルをフォルダごと削除で問題ありません。

●ファイル内容
　・litro_kb.exe：ソフト本体。
　・ltrkb_manual：説明書。（参照時はmanual.htmlを開いてください。）
　・BGMDATA：保存した専用データが入っているところ。
　・BMPFILES：グラフィックデータ。
＊上記フォルダの内容を削除してしまった場合。正常に動作しなくなる恐れがあります。


●動作環境
WindowsXPにて動作確認しています。（自分のPC・VAIO TYPE-S VGN-S94Sで動作確認）
そのほかのOSでは確認していませんが、Windows98、Windows2000でも問題ないかと思います。

●著作権等
このソフトはフリーソフトですが、２次的に配布することは固く禁じます。
（でも、個人で作った音楽編集データの配布等は自由です。）
